## Learn using JWT with Passport authentication

Requires [node.js](https://nodejs.org/en/) and [express.js](https://expressjs.com/) to run

#### Directory Structrue
```
passport-jwt/
    +-- configs
    ¦    +-- passport.js
    +-- routes
    ¦    +-- auth.js
    ¦    +-- users.js
    +-- app.js
    +-- package.json
```

#### Test API
#####  Install dependencies
`$ yarn install` or  `$ npm install`

#####  Start app
`$ yarn start` or `$ npm start`
